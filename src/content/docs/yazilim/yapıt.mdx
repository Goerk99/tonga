Struct Instance -> Yapıt

Bir [Yapı](/tonga/tr/yazilim/yapı) ile oluşturulan [Nesnelere](/tonga/tr/yazilim/nesne) verilen addır.


  Go dili ile örnek
  ```go 
  type User Struct {
    string name
    int age;
  };
  var user1 User; // <- Bu bir yapıt
  ```
