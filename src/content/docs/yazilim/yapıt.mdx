Struct Instance -> Yapıt

Bir [Yapı](/yazilim/yapı) ile oluşturulan [Nesnelere](/yazilim/nesne) verilen addır.


  Go dili ile örnek
  ```go 
  type User Struct {
    string name
    int age;
  };
  var user1 User; // <- Bu bir yapıt
  ```

Tek sözcük şekilinde İngilizce karşılığı yoktur.
